Тестовое задание на адаптивную верстку страницы по макетам.

Требования:
1) Необходимо сверстать используя только grid bootstrap (без лишних компонентов и стилизаций bootstrap.)
2) flexgrid допускается только для блока информации о товаре (обязательная кроссбраузерность для ie 10)
3) Дублирование блоков в коде нежелательно (особенно корзины, поиска и избранного)
4) Код должен проходить валидацию
5) Все ссылки должны быть стилизированы в зависимости от их состояния
6) Код должен быть правильно и красиво оформлен ( соблюдать уровень вложенности элементов, понятное именование классов, использовать комментарии в js )
7) Реализовать мобильное меню в котором с планшетного разрешения будут выводится категории, а с мобильного будут добавляться и пункты меню.
8) В изображениях товара нужно реализовать следующее:
   - дополнительные изображения сделать каруселью до 5шт.
   - по клику на доп. изображение, его нужно подставлять на место основного изображения с соответствующими размерами.
   - по клику на основное изображение, выводить это же изображение во всплывающем окне с большим размером.
   В итоге изображения должны быть трех размеров ( так как в макетах отсутствуют изображения разных размеров, можно использовать для примера свои )
9) Шрифт OpenSans нужно подключить через сервис GoogleFonts, а Lato через css @font-face (шрифт во вложение)
10) По клику на обратный звонок, реализовать всплывашку, в которой форма с полями <имя, емайл, текстовое поле и кнопка отправки>.
11) Реализовать скрытие описания товара. ( ограничить по строкам или по высоте на ваше усмотрение)

Дополнительно можете реализовать:
1) При нажатии на "купить", в информере корзины ( в шапке ) должно выводится количество кликов. Тоже самое нужно реализовать и с избранным (иконка с сердцем)
2) Раскрывающиеся поиск на планшетной версии
3) Ховер на зведочки рейтинга
4) Иконки на сайте в svg формате (можно использовать похожие как в макетах)
